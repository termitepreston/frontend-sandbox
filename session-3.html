Tables

The common tags used in the definition of tables are <table>, <caption>, <tr>, <thead>, and <td>.

Notes on tables

  • The default layout of tables is borderless.
  • Use CSS to give tables borders.
  • The following elements are part of table structure.
    • <caption>
    • <thead>
    • <colgroup>
    • <col>
    • <th>
    • <tbody>
    • <tr>
    • <td>
    • <tfoot>
  • For styling purpose the table elements may be thought of as being put on six superimposed layers.

Column groups and spans

Using column groups we can group one or more adjacent columns. The <colgroup> element must always go after the caption element if there are any and before any <thead>, <tbody>, <tfoot> or <tr> elements.
